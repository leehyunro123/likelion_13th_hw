2주차 세션 정리

BE_황채현 선배님

  1. 초기세팅
    • gitigore 사용: db와 media 같은 git에서 충돌날 수 있는 파일들을 충돌나지 않도록 해줌
  2. 주요 요약
    • HTML 페이지와 view를 연결: HTML페이지를 유저에게 보여주기 위함
    • url 작성해야함
    • 웹페이지를 보기 위해 python manage.py runserver를 터미널에 입력
  3. 다른 페이지 만들기
    • HTML 링크 연결: href(하이퍼텍스트 참조)를 이용하여 페이지간 링크 연결
  4. template 언어
    • 조건문 형태: 2주차 세션 내용 참조
    • 반복문 형태: 2주차 세션 내용 참조
    • 템플릿 필터: 템플릿 변수를 특정 서식에 따라 변환하는 것
    • 템플릿 주석: html 내용 중 주석(숨김)이 필요한 경우 사용 {# 주석처리 할 내용 #}
